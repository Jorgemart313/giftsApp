## Descripción

GiftsApp es una aplicación desarrollada con React que te permite buscar GIFs utilizando la API de Giphy. Ofrece una experiencia de usuario fluida para encontrar GIFs por categorías y la funcionalidad de copiar la imagen del GIF directamente al portapapeles para compartirla fácilmente.

## Características

-   **Búsqueda de GIFs por Categoría**: Explora GIFs introduciendo diferentes términos de búsqueda.
-   **Visualización de Resultados**: Muestra una cuadrícula de GIFs relevantes para cada búsqueda.
-   **Copiar GIF al Portapapeles**: Copia la imagen del GIF directamente al portapapeles con un solo clic, lista para pegar en cualquier aplicación compatible.
-   **Diseño Responsivo**: Interfaz de usuario adaptable a diferentes tamaños de pantalla.

## Tecnologías Utilizadas

-   **React**
-   **TypeScript**
-   **Vite**
-   **Giphy API**
-   **Web Clipboard API**


## Uso

Una vez que la aplicación esté en funcionamiento:

1.  **Introduce una categoría** en el campo de búsqueda.
2.  **Presiona Enter** o haz clic en el botón de búsqueda.
3.  Los GIFs relacionados con tu búsqueda aparecerán en la pantalla.
4.  **Haz clic en cualquier GIF** para copiar su imagen directamente a tu portapapeles. Recibirás una notificación en la consola del navegador si la copia fue exitosa.
